# Code of Conduct

## Our standard

Contributors are expected to communicate clearly, respectfully, and with technical honesty.

## Expected behavior

- Share criticism in a constructive way
- Focus on evidence, tradeoffs, and implementation quality
- Welcome different experience levels and working styles
- Avoid harassment, insults, intimidation, and exclusionary behavior

## Scope

This applies to issues, pull requests, discussions, and any project communication spaces.

## Reporting

Report unacceptable behavior through a private security or maintainer contact path. Project maintainers may remove, edit, or reject contributions that violate these standards.
