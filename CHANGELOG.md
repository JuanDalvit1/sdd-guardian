# Changelog

All notable changes to `sdd-guardian` will be documented in this file.

## [v0.1.0] - 2026-04-02

### Added

- Initial public release of the `sdd-guardian` global Codex skill
- Global install and sync scripts for `%USERPROFILE%\.codex` and `$env:CODEX_HOME`
- Managed `AGENTS.md` snippet for persistent documentation behavior
- Bootstrap templates for `PROJECT.md`, `STATE.md`, architecture docs, ops docs, domain invariants, and ADRs
- Repo scan and documentation drift scripts
- Public README, installation guide, adoption guide, and community health files
- Release packaging script and downloadable ZIP assets for installation

### Changed

- README now points to the latest release for easier installation
- Public repository metadata now supports pinning, topics, and release-driven onboarding
