# Image prompts

If you want to replace the repo visuals with AI-generated artwork later, use prompts like these in GPT image generation or another image workflow.

## Social preview prompt

```text
Use case: infographic-diagram
Asset type: GitHub repository social preview
Primary request: create a premium, editorial tech graphic for a repository called SDD Guardian
Scene/backdrop: dark cinematic gradient background from deep navy to dark teal, no photo elements
Subject: productized concept of long-term engineering context for AI agents, with elegant panels and spatial depth
Style/medium: crisp design-forward infographic, not cartoonish, not generic SaaS stock art
Composition/framing: 1280x640 horizontal composition, strong title area on the left, three structured concept blocks along the bottom, enough negative space for readability
Lighting/mood: premium, calm, precise, systems-thinking mood
Color palette: deep navy, teal, cyan, off-white
Text (verbatim): "SDD GUARDIAN" and "Context that survives delivery."
Constraints: highly legible, no content overflow, no watermark, no fake UI chrome, no purple palette
Avoid: clutter, excessive icons, low-contrast text, generic AI futuristic neon overload
```

## Before / after prompt

```text
Use case: infographic-diagram
Asset type: README comparison graphic
Primary request: create a refined before-versus-after engineering documentation graphic
Scene/backdrop: dark premium background with two large contrasting panels
Subject: left panel represents prompt sprawl and fragile context, right panel represents modular durable project context
Style/medium: clean information design with polished typography
Composition/framing: 1280x720, centered comparison layout, strong arrow transition in the middle
Text (verbatim): "Before / After", "BEFORE", "AFTER", "Prompt sprawl", "Durable context system"
Constraints: professional, sharp, no overflow, no extra icons unless subtle
Avoid: cheap gradients, random symbols, comic style, noisy glow effects
```

## Install flow prompt

```text
Use case: infographic-diagram
Asset type: README onboarding graphic
Primary request: create a four-step premium onboarding graphic for installing a Codex skill globally
Scene/backdrop: dark editorial tech background
Subject: four numbered cards showing clone repo, run install, Codex updated, next PC sync
Style/medium: clean systems diagram with premium typography
Composition/framing: 1280x720, horizontal step flow with spacing and arrows
Constraints: text-first clarity, no overflow, strong alignment, teal and cyan accents, no watermark
Avoid: playful illustration style, generic startup stock look, overstuffed cards
```
